## 🔧 高级用法

### 指定指标类型

可用的指标类型：
- `basic` - 基本财务指标（市盈率、市净率等）
- `profit` - 盈利能力指标
- `growth` - 成长能力指标
- `debt` - 偿债能力指标
- `operation` - 运营能力指标
- `all` - 所有指标（默认）

示例：
```
获取贵州茅台的盈利能力指标
```

### 指定报告期

可用的报告期类型：
- `annual` - 年度报告（默认）
- `quarter` - 季度报告

示例：
```
查询平安银行的季度利润表
```

### 批量查询并保存

```
批量查询000001, 600036, 600519的财务指标，保存为Excel文件
```

## ⚠️ 注意事项

1. **股票代码格式**: 必须是6位数字（如 000001, 600519）
2. **批量查询限制**: 一次最多查询20个股票
3. **数据延迟**: 财务数据可能存在一定延迟
4. **缓存清理**: 缓存数据7天后自动清理
5. **文件大小**: 单个导出文件不超过50MB

## 🐛 常见问题

### Q: 如何确认MCP服务器是否正常运行？
A: 在VS Code中，MCP服务器会自动启动。您可以尝试发送一个简单的查询请求来测试。

### Q: 查询失败怎么办？
A: 检查以下几点：
- 股票代码格式是否正确（6位数字）
- 网络连接是否正常
- 是否在交易时段（数据源可能有限制）

### Q: 如何查看导出的文件？
A: 导出成功后，系统会返回文件的完整路径。您可以直接在文件资源管理器中打开。

### Q: 可以查询港股或美股吗？
A: 当前版本仅支持A股市场。

### Q: 数据更新频率如何？
A: 财务报表数据通常在公司发布后的1-2天内更新。
